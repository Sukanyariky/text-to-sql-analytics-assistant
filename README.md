# 🤖 Text-to-SQL Analytics Assistant

## Overview

This project demonstrates an LLM-powered analytics assistant capable of converting natural language business questions into executable SQL queries.

The application helps business users retrieve analytical insights without writing SQL manually.

---

## Problem Statement

Business users often rely on analysts to answer SQL-based questions.

This project explores how Large Language Models can automate SQL generation and improve data accessibility.

---

## Tech Stack

- Python
- OpenAI / Gemini API
- LangChain
- SQL
- Pandas
- SQLite / MySQL

---

## Features

- Natural language understanding
- SQL generation
- Schema-aware prompting
- Query execution
- Result summarization
